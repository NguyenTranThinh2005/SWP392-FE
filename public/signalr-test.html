SignalR Notification Tester

Kết nối bằng JWT của user nhận thông báo và theo dõi event realtime ReceiveNotification.

Cấu hình kết nối

Hãy dùng JWT của chính user được backend chọn làm người nhận. JWT của user khác vẫn có thể kết nối, nhưng sẽ không nằm trong group user-{receiverUserId}.

Thứ tự test:
1. Login user nhận và dán JWT vào đây.
2. Bấm Connect.
3. Dùng Postman/tab khác gọi API tạo notification.
4. Xem payload xuất hiện bên dưới.
Nếu dùng HTTPS localhost, mở URL API trực tiếp trong trình duyệt và chấp nhận development certificate trước. Live Server cũng phải được backend cho phép CORS.
